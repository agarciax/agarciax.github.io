Projects

Aurora University Lost And Found Item Management System

The AU IMS is a web application created to replace the current excel report system for the AU Police Department, AU Lost And Found. Designed using a RESTful backend server using Node.js w/ the express framework to enable items to be stored persistently into an online databases, and viewed dynamically on a web page for a student/staff member to see.

Wireless Thermostat

A DIY thermostat using the ATmega328P microcontroller to measure the temperature and CO2 levels, alert the surrounding area based on the severity of the gas level, and to display it on an LCD away from harms way using Bluetooth. A buzzer will start to beep faster as the CO level gets higher.
Designed using an analog to digital converter(ADC) to convert readings from the temperature sensor or gas sensor to digital reading, timer/counter to switch between sensors, pulse-width modulation(PWM) to control the voltage of the buzzer, and USART to communicate with the Bluetooth module
